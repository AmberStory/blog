###### hexo发布博客命令
`hexo g`  // 生成静态文件

`hexo deploy`   // 将网站部署到服务器

上面两个命令可以合并成 `hexo g -d`

`hexo g --watch`  // 编译修改过的文件

`hexo n <文件名>`  // 新建文件

`hexo server`   // 启动server预览本地文件

hexo官方文档：https://hexo.io/zh-cn/docs/commands#generate
